# Useless OIIAI Cat

一款完全沒用的 macOS SwiftUI app：只播放去掉綠幕的 OIIAI 貓 GIF 和迷因音效。

## Assets

- `assets/oiiai.gif`: 原始素材
- `assets/oiiai-transparent.gif`: 去掉綠幕後的 app 內播放版本
- `assets/W&W - OIIA OIIA (Spinning Cat) [IxX_QHay02M].mp3`: 自動 loop 音效

## Run

```sh
swift run UselessOiiaiCat
```

## Build

```sh
swift build
```
